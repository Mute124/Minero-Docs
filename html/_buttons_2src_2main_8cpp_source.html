55 -- ButtonR --
56
57 The ButtonR class creates a rectangle-shaped static-size button, window-resized-not-responsive button.
58
59 The button size is computed from the font size and length of the text;
60 the text is automatically centered on the button.
61 See mutators to modify the button’s position, the text’s position in the button, and the button’s size.
62

66

80
81 The button size is computed from the font size and length of the text;
82 the text is automatically centered on the button.
83 See mutators to modify the button’s position, the text’s position in the button, and the button’s size.
84

88
89 The button's shadow can be turn on and off.
90 The shadow size is computed from the font size and length of the text;
91 modifying the button's size, we also modify the shadow's size.
92 See mutators for modifying specifically the shadow's size and position.
93
94 The default font is raylib font.
