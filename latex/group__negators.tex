\doxysubsubsubsubsubsubsection{Detailed Description}
~\newline


The function templates {\ttfamily not1} and {\ttfamily not2} are function object adaptors, which each take a predicate functor and wrap it in an instance of {\ttfamily unary\+\_\+negate} or {\ttfamily binary\+\_\+negate}, respectively. Those classes are functors whose {\ttfamily \doxylink{raylib_2src_2external_2_a_n_g_l_e_2_g_l_e_s2_2gl2ext_8h_ae60229696c2ec17c9e00af4899bf738e}{operator()}} evaluates the wrapped predicate function and then returns the negation of the result.

 The call to {\ttfamily find\+\_\+if} will locate the first index (i) of {\ttfamily v} for which {\ttfamily !(v\mbox{[}i\mbox{]} \texorpdfstring{$>$}{>} 3)} is true.

The not1/unary\+\_\+negate combination works on predicates taking a single argument. The not2/binary\+\_\+negate combination works on predicates taking two arguments.
