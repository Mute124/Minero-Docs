Base class for constructing a {\itshape safe} unordered container type that tracks iterators that reference it. 

The class template \+\_\+\+Safe\+\_\+unordered\+\_\+container simplifies the construction of {\itshape safe} unordered containers that track the iterators that reference the container, so that the iterators are notified of changes in the container that may affect their operation, e.\+g., if the container invalidates its iterators or is destructed. This class template may only be used by deriving from it and passing the name of the derived class as its template parameter via the curiously recurring template pattern. The derived class must have {\ttfamily iterator} and {\ttfamily const\+\_\+iterator} types that are instantiations of class template \doxylink{class____gnu__debug_1_1___safe__iterator}{\+\_\+\+Safe\+\_\+iterator} for this container and {\ttfamily local\+\_\+iterator} and {\ttfamily const\+\_\+local\+\_\+iterator} types that are instantiations of class template \doxylink{class____gnu__debug_1_1___safe__local__iterator}{\+\_\+\+Safe\+\_\+local\+\_\+iterator} for this container. Iterators will then be tracked automatically.
