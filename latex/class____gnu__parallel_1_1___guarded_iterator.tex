class \+\_\+\+\_\+gnu\+\_\+parallel\+::\+\_\+\+Guarded\+Iterator$<$ \+\_\+\+RAIter, \+\_\+\+Compare $>$}
\+\_\+\+Iterator wrapper supporting an implicit supremum at the end of the sequence, dominating all comparisons. 

The implicit supremum comes with a performance cost.

Deriving from \+\_\+\+RAIter is not possible since \+\_\+\+RAIter need not be a class.
