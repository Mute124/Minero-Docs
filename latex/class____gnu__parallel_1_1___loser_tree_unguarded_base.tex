Base class for unguarded \doxylink{class____gnu__parallel_1_1___loser_tree}{\+\_\+\+Loser\+Tree} implementation. 

The whole element is copied into the tree structure.

No guarding is done, therefore not a single input sequence must run empty. Unused \+\_\+\+\_\+sequence heads are marked with a sentinel which is \texorpdfstring{$>$}{>} all elements that are to be merged.

This is a very fast variant.
