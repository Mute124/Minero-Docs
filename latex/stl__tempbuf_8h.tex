This function tries to obtain storage for {\ttfamily \+\_\+\+\_\+len} adjacent Tp objects. The objects themselves are not constructed, of course. A pair$<$$>$ is returned containing {\itshape the buffer s address and capacity (in the units of \doxylink{raylib_2src_2external_2_a_n_g_l_e_2_g_l_e_s2_2gl2ext_8h_ae60229696c2ec17c9e00af4899bf738e}{sizeof(\+\_\+\+Tp)}), or a pair of 0 values if no storage can be obtained.} Note that the capacity obtained may be less than that requested if the memory is unavailable; you should compare len with the .second return value.

Provides the nothrow exception guarantee.
